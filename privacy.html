FluxState Pin Publisher — Privacy Policy

Last updated: June 2026

Overview

FluxState Pin Publisher is a private automation tool used solely by its owner to post product listing pins to their own Pinterest business account. This app is not affiliated with or endorsed by Pinterest.

Pinterest API Usage

This app uses the Pinterest API to create pins on the owner's own Pinterest business boards. It does not collect, store, resell, or redistribute any Pinterest-derived data or Pinterest content to third parties.

Data Collection

This app does not collect any personal data from users. It is a single-user private tool with no public-facing interface.

Pinterest API Data

Any data retrieved from the Pinterest API is used solely to facilitate posting pins to the owner's own account. No Pinterest-derived data is shared with or sold to third parties.
